== Installation ==

To use the Salvador AI image plugin you will need an active OPENAI account, and active Dall-e API key.
- 1. Obtain a Dall-e license key here https://beta.openai.com/signup
- 2. To get full access to the features of Salvador and its modifiers please upgrade to PRO (or use the free version)

To install a plugin on WordPress, follow these steps:

Log in to your WordPress admin dashboard.
- 1. In the left-hand menu, click on "Plugins" and then click on "Add New".
- 2. In the search box, type Salvador AI and press enter.
- 3. Locate the plugin in the search results and click on the "Install Now" button.
- 4. After the plugin has finished installing, click on the "Activate" button to activate the plugin.
- 5. Once the plugin has been installed and activated, you can access its settings and configuration options by going to a post or page, clicking the featured image or insert an image selection, and clicking on the Salvador AI Images text.

Alternatively, you can also install a plugin by uploading the plugin files to your WordPress website. To do this, follow these steps:

- 1. Download the plugin files from the plugin's website or from the WordPress plugin repository.
- 2. In your WordPress admin dashboard, go to the "Plugins" menu and click on "Add New".
- 3. Click on the "Upload Plugin" button and select the plugin files that you downloaded in step 1.
- 4. After the plugin has finished uploading, click on the "Activate" button to activate the plugin.
- 5. Once the plugin has been activated, you can access its settings and configuration options in the same way as described above.

DALL-E is named after the artist Salvador Dali and the Pixar character WALL-E. It is a generative model that is able to create images based on text descriptions, using a combination of natural language processing and computer vision techniques. DALL-E is able to generate these images because it has been trained on a large dataset of images and text descriptions, which allows it to learn the relationship between words and visual features.

= How does AI image generation work ? =

AI image generation is a type of AI technology that uses machine learning algorithms to create new images based on a set of input images. This is typically done using a deep learning model trained on a large dataset of images. The model is able to learn the patterns and features of the input images and use that knowledge to generate new images that are similar in style and content to the input images.

The process of AI image generation typically involves several steps. First, the input images are preprocessed to extract the relevant features and prepare them for input to the model. Next, the model is trained on the input images using a variety of techniques, such as supervised learning, unsupervised learning, or generative adversarial networks (GANs).

Once the model is trained, it can be used to generate new images by providing it with a set of input images and a set of constraints or conditions that the generated images should satisfy. For example, the model could be asked to generate images of a specific object or scene, or images that have a specific color or texture. The model uses the input images and the constraints to generate new images that are similar to the input images but also satisfy the constraints.

AI image generation has many potential applications, such as creating new images for use in advertising, art, or design. It can also be used to enhance or improve existing images, or to generate images for use in machine learning algorithms. Overall, AI image generation is a powerful tool for creating and manipulating images using AI technology.


= What are the legal issues with AI image generation ? =

There are several legal issues that may arise in relation to AI image generation. Some of the key issues include:

 - Copyright infringement: AI image generation may involve the creation of images that are based on or derived from existing images, which may infringe the copyright of the original creators of those images. In some cases, this may require obtaining permission from the original creators or obtaining a license to use the original images.

 - Trademark infringement: AI image generation may also involve the creation of images that use or incorporate trademarks, logos, or other branding elements, which may infringe the trademark rights of the owners of those marks. In such cases, it may be necessary to obtain permission from the trademark owners to use their marks in the generated images.

 - Right of publicity: AI image generation may involve the creation of images that include the likenesses of real people, which may infringe their right of publicity. In such cases, it may be necessary to obtain permission from the individuals depicted in the generated images to use their likenesses.

 - Privacy: AI image generation may also raise privacy concerns, especially if the generated images include personal information or sensitive data. In such cases, it may be necessary to take steps to protect the privacy of individuals depicted in the generated images, such as by obscuring their faces or other identifying features.

Overall, it is important to consider these legal issues when using AI image generation and to take appropriate steps to avoid infringing the rights of others and to protect the privacy of individuals.
